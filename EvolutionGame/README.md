# Adapt or Die Out

Компьютерная версия настольной игры **«Эволюция»** на движке Unity (C#).
Курсовой проект по дисциплине «Основы программирования».

## Описание

«Adapt or Die Out» — цифровая адаптация настольной карточной стратегии «Эволюция»,
в которой игроки развивают виды животных, наделяют их свойствами и конкурируют
за ресурсы. Программа поддерживает многопользовательский режим до 4 игроков
на одном устройстве и режим игры с ботами.

## Возможности

- 🃏 Полная колода из 84 карт с 19 типами свойств
- 👥 От 2 до 4 игроков на одном устройстве
- 🤖 Игра против ИИ (от 1 до 3 ботов)
- 🔄 Полный игровой цикл: развитие → определение кормовой базы → питание → вымирание
- 🎵 Настройки громкости музыки и звуковых эффектов
- 🖥️ Полноэкранный и оконный режим

## Системные требования

**Минимальные:**
- ОС: Windows 10/11, macOS 10.12+, Linux
- Процессор: Intel Core i3 2.0 ГГц или аналогичный
- ОЗУ: 4 ГБ
- Место на диске: 500 МБ
- Разрешение экрана: от 1280×720

## Технологии

- **Unity** 2022.3 LTS
- **C#** (.NET Standard 2.1)
- **Unity Test Framework** для юнит-тестов


## Запуск из исходников

1. Установить Unity Hub и Unity 2022.3 LTS.
2. Клонировать репозиторий:
   ```bash
   git clone https://github.com/<username>/EvolutionGame.git
   ```
3. Открыть папку `EvolutionGame` через Unity Hub.
4. Открыть сцену `Assets/Scenes/MainMenu.unity` и нажать ▶ Play.

## Сборка релизной версии

В Unity: `File → Build Settings → Build`. Целевая платформа — Windows x64.

## Запуск тестов

`Window → General → Test Runner → EditMode → Run All`.
